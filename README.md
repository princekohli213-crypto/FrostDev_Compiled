## 🛠 Prerequisites

If you plan to run the source code or build the Tauri application yourself, ensure you have the following components installed on your environment:

1.  **Node.js & npm:** Required for managing frontend dependencies and running scripts.
2.  **Rust & cargo:** The backend backbone for Tauri. You can install it via [rustup.rs](https://rustup.rs/).
3.  **Visual Studio Build Tools (Windows Only):** Essential C++ build tools required by Rust to compile the native binaries. Select the "Desktop development with C++" workload during installation.

---

## 🚀 Getting Started

Once your environment meets all the prerequisites above, follow these steps to run the application in development mode:

1. Clone or download the source code repository.
2. Open your terminal in the project root directory.
3. Install the frontend dependencies:
   ```bash
   npm install
4. Run The App
   ```bash
   npm run tauri dev
